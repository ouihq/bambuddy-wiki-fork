# Materials Required

This page is a practical BOM for the current integrated SpoolBuddy setup.

---

## :material-format-list-bulleted: Core BOM

| Category | Item | Qty |
|----------|------|-----|
| Compute | Raspberry Pi 4B (4GB+ recommended; 2GB for companion-only mode) or Raspberry Pi 5 | 1 |
| Display | LAFVIN 7 inch HDMI LCD 1024x600 Capacitive Touch IPS Display | 1 |
| NFC | PN5180 NFC Reader | 1 |
| Scale ADC | SparkFun Qwiic NAU7802 | 1 |
| Scale | Load Cell - 5kg Single Point | 1 |
| Tags | NTAG213/215/216 stickers/cards | As needed |
| Power | Official or high-quality Pi PSU | 1 |
| Power input | Angled 90° USB-C connector | 1 |
| Storage | microSD card (good quality) | 1 |

Display note: this display kit includes Raspberry Pi adapter pieces (angled HDMI/micro USB style adapters), which simplifies mounting/cabling.

## :material-soldering-iron: Wiring and Build Supplies

| Item | Why |
|------|-----|
| Soldering iron | Required for wiring and module connections. |
| Flux (optional) | Makes solder flow easier and improves joint quality. |
| Copper wool / tip cleaner | Keeps the soldering iron tip clean for reliable soldering. |
| Isopropyl alcohol (optional) | Cleans sticky flux residue after soldering. |
| Heat-shrink tubing | Keeps wires together and provides insulation/strain relief. |
| Flux-core solder wire | Recommended solder wire for easier, cleaner joints. |
| Solder fume extractor (optional) | Helps keep soldering fumes away from your breathing zone. |
